Event Trio Website

A modern, elegant event planning website designed for Event Trio, a Rwanda-based brand focused on delivering premium, stress-free event experiences.

This platform is built to present the company’s services, pricing packages, portfolio, and team while providing a smooth way for clients and partners to connect.

✨ Overview

Event Trio’s identity is centered around simplicity, elegance, and perfection. The website reflects this through a luxury black and gold design system, clean layouts, and emotionally-driven content that highlights the value of professional event planning.

🎯 Features
Service presentation (Event Planning, Styling, Coordination, etc.)
Pricing packages with clear structure
Portfolio/gallery of past events
Team introduction section
Contact & booking integration
Fully responsive modern UI

🎨 Design System
Colors: Deep black, gold accents, soft neutrals
Style: Minimal, premium, visually rich
UX Focus: Smooth navigation, clear hierarchy, emotional storytelling
🛠️ Tech Stack

React + Vite
TailwindCSS
Node.js / Express
MySQL

📍 Purpose

The goal of this project is to create a strong digital presence for Event Trio, helping them:

Attract high-quality clients
Clearly communicate their value
Build trust through design and presentation
